笔记一 Android compile command

1. setup environment and supply function ./build/envsetup.sh

1.1 lunch----lunch menu and lunch the board

1.2 m----在当前代码树的顶层路径执行make命令，不管哪层路径都可以执行此命令

1.3 mm----在当前路径下执行make命令，当前路径如果是顶层，相当于全编，不包括它们的依赖文件

如果使用mm方式编译，就会删除system.img、ramdisk.img、userdata.img等IMG镜像文件

1.4 mmm----编译指定路径的全部文件，不包括它们的依赖文件（mmm packages/apps/Dialer/），用Tab键可以补全路径名称。

例子 mmm -B packages/apps/Dialer/ 单独编译某个模块mmm packages/apps/Dialer:Dialer

1.5 hmm ----print the usage of the envsetup supply fuction
